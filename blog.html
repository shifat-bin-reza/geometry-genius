What is DOM and what is the purpose of DOM?

Answer: DOM stands for Document Object Model. DOM helps us to manipulate HTML elements as well as add functionallity to HTML elements.

How will you select HTML elements using DOM? Name the DOM methods.

Answer: There are five methods to select HTML elements using DOM. Here are them:

1. getElementsByTagName()

2. getElementsByClassName()

3. getElementById()

4. querySelector()

5. querySelectorAll()

What is event bubble?

Answer: Event bubble basically happens when an element receives an event and it bubbles up to its parent util it reaches to its root element. This is a default behavior of events inside element and to stop this we can to stopPropagation() method.

What is a callback function and why will you use it?

Answer: A callback function is a function that receives an argument from another function and is subsequently called from inside the outer function to finish a task or activity.
